# Проект GUI приложений на Tkinter

## Обзор проекта
Этот проект содержит два графических приложения, созданных с использованием фреймворка Tkinter для Python с расширенными возможностями обработки изображений через библиотеку Pillow.

## Технологический стек

### Основные технологии
- **Python 3.8+** - основной язык программирования
- **Tkinter** - стандартный фреймворк для GUI
- **Pillow (PIL)** - библиотека для обработки изображений

### Ключевые компоненты Tkinter
- `tk.Tk` - главное окно приложения
- `Label` - виджет для отображения текста и изображений
- `Frame` - контейнер для компоновки элементов
- `place()` - менеджер геометрии для точного позиционирования

### Обработка изображений через Pillow
- `Image.open()` - загрузка и обработка изображений
- `Image.resize()` - масштабирование с фильтром LANCZOS
- `ImageDraw` - инструменты для рисования
- `ImageTk.PhotoImage` - конвертация изображений для Tkinter
- Поддержка альфа-каналов и прозрачности

## Описание приложений

### planet2.py - Приложение для отображения изображений
**Возможности:**
- Качественный рендеринг изображений со сглаживанием
- Масштабирование с фильтром LANCZOS
- Текстовые метки с настройкой шрифтов
- Абсолютное позиционирование элементов
- Обработка ошибок файловых операций

**Основные компоненты:**
- Класс `MainWindow` для управления приложением
- Конвейер загрузки и предобработки изображений
- Настройка адаптивной геометрии окна

### profile2.py - Интерфейс профиля пользователя
**Расширенные возможности:**
- Обработка круглых аватарок
- Пользовательское оформление фона
- Многострочный текст с переносом слов
- Сложная компоновка интерфейса
- Профессиональный стиль UI
# Приложение для отображения изображений
python planet2.py

# Приложение профиля пользователя
python profile2.py
